


# Demo

[GO TO APP](https://listus.firebaseapp.com)

# To install
```bash
	npm install
```

# To lint
```bash
	npm run lint
```

# To test

```bash
	npm test
```

or

```bash
	npm run test:watch
```

# To dev start
```bash
	npm start
```

And goto [http://localhost:8080](http://localhost:8080)

# To dist
```bash
	npm run dist
```
